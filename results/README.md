# Results Directory

This directory stores experimental results, visualizations, and figures.

**⚠️ Large image/video files are NOT tracked by Git.**

## Structure

```
results/
├── eda_*.png              # EDA visualizations
├── phase1_*.png           # Phase 1 results
├── phase2_*.png           # Phase 2 results
└── experiments/           # Experiment-specific results
    └── EXP-YYYYMMDD-NN/
```

## File Naming

- `eda_mri_sample.png` - EDA visualizations
- `phase2_loss_curve_exp01.png` - Training curves
- `phase2_parameter_comparison_exp01.png` - Prediction results
